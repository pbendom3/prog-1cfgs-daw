BONUS: el operador ternario en Java

El operador ternario

Existe un formato de condición adicional, que permite comprobar si se cumple una condición y devolver un cierto valor según si dicha condición se cumple o no. Es lo que se conoce como el "operador ternario" ( ? ).

variable = condicion ? resultado_si_cierto : resultado_si_falso

Es decir, se indica la condición seguida por una interrogación, después el valor que hay que devolver si se cumple la condición (true), a continuación un símbolo de "dos puntos" (:) y finalmente el resultado que hay que devolver si no se cumple la condición (false).

Es frecuente emplearlo en asignaciones (aunque algunos autores desaconsejan su uso porque puede resultar menos legible que un "if"), como en este ejemplo:

x = a == 10 ? b*2 : a ;

En este caso, si "a" vale 10, la variable "x" tomará el valor de b*2, y en caso contrario, tomará el valor de "a". Esto también se podría haber escrito de la siguiente forma, más larga pero más legible:

if (a == 10){
    x = b*2;
}else{
    x = a;
}

Podemos concatenar ternarias en ambos resultados. Por ejemplo,

- si lo hacemos en la parte del true:

int a = 10, b = 10, x;
x = a == 10 ? b == 10 ? 100 : 1 : a ; //100

- si lo hacemos en la parte del false:

int a = 1, b = 10, x;
x = a == 10 ? b*2 : b == 10 ? 100 : a; //100

Siguiendo este formato, podemos concatenar todas las ternarias que necesitemos:

int a = 1, b = 2, c = 3;

boolean condicion = a == 1 ? true : false;

if (condicion) System.out.println("a vale 1");
else System.out.println("a no vale 1");

condicion = a == 3 ? true :
            b == 3 ? true :
            c == 3 ? true : false;

if (condicion) System.out.println("Alguna variable vale 3");
else System.out.println("Ninguna variable vale 3");

a = a != 0 ? 2 : 3; //a vale 2

b = a == c ? 2 : 1; //b vale 1

Otro uso común es formatear frases a mostrar por pantalla que tengan 2 posibles salidas:

System.out.println("La letra A ha aparecido " + cantidad + " " +(cantidad == 1 ? "vez" : "veces"));
