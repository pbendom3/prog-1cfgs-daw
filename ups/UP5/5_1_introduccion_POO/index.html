5.1. Introducción a la POO

Durante la década de los 80 y principios de los 90, el paradigma de programación estructurada (la que hemos visto hasta ahora) fue claramente el dominante a la hora de realizar programas más o menos complejos. Sin embargo, dicho paradigma presentaba algunas carencias a la hora de abarcar problemas complejos. 


La programación orientada a objetos (POO) supone un nuevo paradigma frente a la programación estructurada o modular. En esta, los datos y operaciones se encuentran separados (por un lado las variables y por otro las instrucciones), y los métodos se van pasando de unos a otros los datos que necesitan, mediante parámetros. En la POO, los datos y métodos se agrupan, formando así un bloque que contiene todo que se denomina clase. Así, parece más lógica y sencilla la relación entre los datos e instrucciones que componen un mismo programa. 

Por ejemplo, si tuviera un programa para manipular un triángulo...

  • En la programación modular, para calcular su área, necesitaría recoger y guardar dos valores en variables (base y altura), y luego pasárselas al método que calculara el área y nos devolverá el resultado para imprimirlo:
public class Triangulo {

    public static void main(String[] args) {
        double base = leerValor("Introduce la base del triángulo: ");
        double altura = leerValor("Introduce la altura del triángulo: ");
        double area = calcularArea(base, altura);
        mostrarResultado(area);
    }

    public static double leerValor(String mensaje) {
        Scanner scanner = new Scanner(System.in);
        System.out.print(mensaje);
        return scanner.nextDouble();
    }

    public static double calcularArea(double base, double altura) {
        return (base * altura) / 2;
    }

    public static void mostrarResultado(double area) {
        System.out.println("El área del triángulo es: " + area);
    }
}
  • Si traducimos lo anterior a la POO, necesitaríamos 2 variables (o características) asociadas a la clase Triangulo (a partir de ahora, llamadas ATRIBUTOS), junto a los métodos necesarios para calcular el área:
public class Triangulo2 {

    double base;
    double altura;

    public Triangulo2(double base, double altura) {
        this.base = base;
        this.altura = altura;
    }

    public double calcularArea() {
        return (base * altura) / 2;
    }

    public void mostrarResultado() {
        System.out.println("El área del triángulo es: " + calcularArea());
    }
}

 Y desde main:

public class Main {
    public static void main(String[] args) {

        Triangulo2 triangulo = new Triangulo2(5, 2);
        System.out.println("Has introducido base = " + triangulo.base + " y altura = " + triangulo.altura);
        triangulo.mostrarResultado();
    }

}
