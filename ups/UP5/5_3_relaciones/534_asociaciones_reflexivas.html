5.3.4. Asociaciones reflexivas

Una asociación reflexiva ocurre cuando una clase tiene una relación con ella misma. En otras palabras, un objeto de la clase puede estar asociado con otro objeto de la misma clase.

Un ejemplo...

Imaginemos un Empleado que tiene un Supervisor, y el Supervisor también puede ser un Empleado. Es decir, un Empleado puede ser supervisado por otro Empleado, creando una relación reflexiva entre objetos de la misma clase.

class Empleado {
    private String nombre;
    private String puesto;
    private Empleado supervisor; //RELACIÓN REFLEXIVA

    public Empleado(String nombre, String puesto) {
        this.nombre = nombre;
        this.puesto = puesto;
        this.supervisor = null;  //No tiene supervisor inicialmente
    }

    //Método para asignar supervisor
    public void asignarSupervisor(Empleado supervisor) {
        this.supervisor = supervisor;
    }

    //Método para obtener supervisor
    public Empleado getSupervisor() {
        return supervisor;
    }

    public void mostrarInfo() {
        System.out.println("Empleado: " + nombre + " - Puesto: " + puesto);
        if (supervisor != null) {
            System.out.println("Supervisor: " + supervisor.nombre);
        } else {
            System.out.println("Este empleado no tiene supervisor.");
        }
    }
}

Con el siguiente Main:

public class Main {
    public static void main(String[] args) {

        Empleado empleado1 = new Empleado("Juan", "Desarrollador");
        Empleado empleado2 = new Empleado("Ana", "Gerente");
        Empleado empleado3 = new Empleado("Pedro", "Supervisor");

        empleado1.asignarSupervisor(empleado2);  //Juan es supervisado por Ana
        empleado2.asignarSupervisor(empleado3);  //Ana es supervisada por Pedro

        empleado1.mostrarInfo();
        empleado2.mostrarInfo();
        empleado3.mostrarInfo();

    }
}

Ejercicio

Modifica el ejemplo anterior para que, además de un supervisor, los empleados puedan tener compañeros de trabajo que también sean de tipo Empleado.
